# Moje kočička 🐱

Jednoduchá prohlížečová hra ve stylu Tamagotchi – staráš se o vlastní kočku.

## Jak hru spustit

Dvakrát klikni na soubor **`index.html`** – otevře se v prohlížeči. To je vše,
žádná instalace není potřeba. Funguje na počítači i na mobilu.

## Jak se hraje

Na začátku si kočku pojmenuješ. Pak se o ni staráš pomocí tlačítek:

| Tlačítko | Co dělá |
|----------|---------|
| 🍗 **Nakrmit** | Doplní **sytost** (kočka má časem hlad) |
| 🧶 **Hrát** | Zvýší **štěstí** (ale ubere trochu energie) |
| 💗 **Pohladit** | Zvýší **štěstí** |
| 🧼 **Uklidit** | Obnoví **čistotu** záchodku |
| 😴 **Spát / Probudit** | Kočka spí a dobíjí si **energii** |

Čtyři ukazatele (sytost, štěstí, energie, čistota) časem **samy klesají**, takže
o kočku musíš pravidelně pečovat. Když některý spadne nízko, kočka dá najevo,
co potřebuje. Cílem je udržet ji šťastnou a spokojenou. 😻

## Ukládání

Hra se automaticky ukládá v prohlížeči (`localStorage`). Když ji zavřeš a později
otevřeš, kočka tam bude i s tím, jak se o ni mezitím nikdo nestaral.
Tlačítkem **Nová hra** můžeš začít znovu.
